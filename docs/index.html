开始使用

三步安装，即刻开启 AI 翻译

无需后端服务，无需注册插件账号。下载代码、加载扩展、配置 API，三分钟完成。

01

下载代码

克隆项目仓库到本地，或直接下载 ZIP 压缩包解压。

git clone https://github.com/iwsuccess/ai-translate-extension.git
02

加载扩展

打开 Chrome 浏览器，访问 chrome://extensions/， 开启右上角「开发者模式」，点击「加载已解压的扩展程序」并选择项目文件夹。

03

配置 API

点击浏览器工具栏的扩展图标，在设置中填入你的 API Key、Base URL 和模型名称。 点击「测试连接」验证配置，保存后即可使用。支持 DeepSeek / OpenAI / Moonshot 等所有兼容服务。
